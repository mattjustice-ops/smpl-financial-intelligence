# Layout hierarchy

## Scan order (every content slide)

1. **Section label** — green band, operating context (GTM, ARR, CASH).
2. **Title + subtitle** — one line outcome; subtitle = story chain position.
3. **KPI strip** — max 4 cards; largest numeric emphasis.
4. **Primary visual** — one chart OR one table dominates the visual zone (~60% height).
5. **Commentary panel** — orange header when split layout (`marketing_source`, `executive_scorecard`).
6. **Footer band** — max 3 bullets OR callouts; never overlap visual zone.

## Anti-patterns (never on board band)

- Full Excel grids as “slides”
- Dual charts side-by-side
- Chart + full table on same story slide (table → appendix)
- Pie / gauge / 3D chart types

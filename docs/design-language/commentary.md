# Executive commentary

## Frame (MD&A and story slides)

1. **What changed** — operational fact, one sentence
2. **Why** — driver / root cause
3. **So what** — strategic implication
4. **Action** — recommendation with owner when possible

## Tone

- Board-ready, not analyst memo
- Favorable / unfavorable / watch via KPI tone and callout `kind`
- No hedging stacks (“might potentially…”)
- Max **3 footer bullets** on operating slides

## Commentary panel (orange header)

Used on:

- `executive_scorecard` — “Key takeaways”
- `marketing_source` — “Channel signals”
- `story_slide` — section-specific heading from builder

## Wins / risks / actions

`risk_matrix` slide uses structured `CalloutBlock` — not freeform paragraph walls.

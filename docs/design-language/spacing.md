# Spacing

## Canvas

- 16:9 — 13.333″ × 7.5″
- Side margins: 0.75″
- Content width: 11.83″

## Vertical zones (fixed)

| Zone | Top (in) | Height (in) | Role |
|------|----------|-------------|------|
| Title | 0.28 | ~1.07 | Label + headline |
| KPI | 1.42 | 0.72 | KPI strip |
| Visual | 2.22 | 2.82 | Chart or table |
| Footer | 5.12 | 1.15 | Bullets / callouts |
| Safe bottom | — | 6.82 cap | Watermark + footnote |

## Rhythm rules

- **KPI gap:** 0.14″ between cards
- **Whitespace before density:** Prefer dropping a row over shrinking chart below 2.2″ height
- **Marketing split:** Table left ~65%, commentary right ~35% (fixed in renderer)
- **Executive scorecard:** KPI row full width; table + takeaways split below

Spacing is **not** computed per slide from content volume — overflow triggers appendix, not compression heuristics.
